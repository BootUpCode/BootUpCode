## Hello, and welcome!

I'm currently improving my ability to write applications in Java and C#.

My latest project is TimeForBattle, a management tool for Game Masters running tabletop games. It's written in C# and XAML using the MAUI framework, and relies on a local SQL database. It runs on PC, Android and IOS, so it's easy to use it while you're running a tabletop game! Check it out for yourself!
