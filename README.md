## Hello, and welcome!

Hey, I'm Martijn! I create small and large programming projects for fun in my spare time. My goal is to improve my ability to write applications in Java and C#, and to explore other languages and frameworks.

### My Projects

Check out my latest small project: [DuckTacToe](https://bootupcode.github.io/DuckTacToe/), inspired by the classic two-player game TicTacToe and the chess variant Duck Chess. It's written in JavaScript using the React library, and hosted on GitHub Pages!

My latest big project is [TimeForBattle](https://github.com/BootUpCode/TimeForBattle), a management tool for Game Masters running tabletop games. It's written in C# and XAML using the MAUI framework, and relies on a local SQL database. It runs on PC, Android and IOS, so it's easy to use it while you're running a tabletop game! The most recent release includes a standalone executable file for Windows, so it's easy to setup and try for yourself.
